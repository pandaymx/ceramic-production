# 实现未来7天产量预测及对比分析功能计划

## 任务目标

在现有陶瓷生产管理系统中实现：

1. **未来7天产量预测值** - 默认显示未来7天预测
2. **历史测试集对比** - 对比过去实际产量与预测产量
3. **季节性区分** - 旺季（3-5月、9-11月）与淡季（1-2月、7-8月）
4. **误差控制** - 误差控制在150以内

***

## 技术实现细节

### 季节性判断逻辑

```python
def get_season_type(month):
    """判断旺淡季"""
    if month in [3, 4, 5, 9, 10, 11]:
        return "peak"  # 旺季
    else:
        return "off"   # 淡季
```

### 误差控制策略

1. **自动检测**：当RMSE > 150时提示需要调整
2. **模型切换**：高峰期自动切换到SARIMA模型
3. **数据清洗**：去除异常值后再训练
4. **集成预测**：ARIMA + LSTM 加权平均
